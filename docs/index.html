Get Started

Install

  • npm i -g mogara
  • or download latest mogara-win.exe

Create a index.mg file and paste:

# Hello World

Now run this command:

mogara index.mg

Styling

Mogara HTML
*italic* italic tag
** Bold ** Bold tag
_Underline_ Underline
~Strike~ strike
^Super Script^ Superscript tag
__SubScript__ Subscript tag
Example :- 
*Italic*
**Bold**
_Underline_
~Strike~
^Super Script^
__Sub Script__

Pre & Code

Mogara HTML
``` code ``` pre tag
`code` code tag
Example :- 
`const a = 10;`

Links and images

Mogara HTML
[text](href) anchor tag
![alt](src) image tag
Example :- 
[Visit example.com](https://example.com)

Ordered and Unorder lists

Mogara HTML
( -or*or+ ) unorder list tag
( 1. 2. 3. ... ) order list tag
Example :- 
- Delhi
+ Mumbai
* Chennai

1. Apple
2. Mango
3. Banana

Div tag container

Mogara HTML
<? Mogara Code Goes Here ?> div tag
<? Mogara Code Goes Here ?>(class="wrapper") div tag with attributes like class, id, styles, etc
Example :- 
<?
This is a div tag
?>

<?
This is a div tag with class attribute
?>(class="wrapper")

Css Styling

Internal css

  • In mogara you can write any css code in between <css style code >
Example:- 
<css
body{
    color: red;
}
>

External css

  • In mogara you can write the path to .css in <css: path/to/css >
Example:- 
    <css: path/to/css >

Inline css ( Only for div tag )

  • In mogara you can write inline css for divs in circular braces ()
Example:- 
<?
[example.com](example.com)
?>(style="background-color: black;")

Embedded JS

Example:- 

<js
function sayHey(name){
    return 'Hey! ' + name;
}
>

<js sayHey("John Deo") >
  • In mogara you can also use .js files code.
Example:- 
<js: path/to/js >
